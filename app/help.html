Работа с графом

Вершины

Для добавления вершины в режиме вершин(режим по-умолчанию) используйте ЛКМ или с помощью пункта контекстного меню(ПКМ) - добавить вершину здесь, в любом режиме. Для перемещения вершины удерживайте ЛКМ на ней и двигайте мышью. Удаление вершин осуществляется с помощью пункта контекстного меню(ПКМ) - удалить или в режиме удаления при щелчке ЛКМ на вершине. Изменение имени(текста) вершины возможно с помощью пункта контекстного меню(ПКМ) - переименовать.

Ребра

Добавление ребра осуществляется с помощью пункта контекстного меню(ПКМ) - добавить ребро на исходной вершине или в режиме ребер, соединяя вершины ЛКМ, кликнув на исходную, а затем на конечную вершины. Удаление ребер осуществляется в помощью пункта контекстного меню(ПКМ) - удалить или в режиме удаления при щелчке ЛКМ на ребре. Изменение веса ребра возможно с помощью пункта контекстного меню(ПКМ) - изменить.

Загрузка, сохранение

Программа работает с графами формата GML. Для загрузки графа из файла(с клиента) воспользуйтесь пунктом меню Файл - открыть. Созданный или измененный граф можно сохранить с помощью пункта меню Файл - сохранить. В диалоговом окне можно изменить имя скачаемого файла(по-умолчанию новый граф).

Алгоритмы

Для использования какого-либо алгоритма используйте пункт меню Алгоритмы. Доступные алгоритмы:

  • Компоненты сильной связности
  • Расстояния в ориентированном графе
  • Поиск минимального пути
  • Эйлеров цикл
  • Эйлерова цепь
  • Поиск минимального остовного дерева
  • Степени вершин

Матрицы

Для представления структуры графа в виде матрицы(матрицы смежности) выберете пункт меню Инструменты - матрица смежности.

Работа с программой

Масштаб рабочей области

Для масштабирования рабочей области воспользуйтесь колесиком мыши(прокрутка вверх/вниз). Также можно использоваь инструмент лупа(пункт меню Инструменты - лупа), кликая ЛКМ на рабочей области. Для однократного увеличения/уменьшения масштаба используйте пункты меню Вид - увеличить/уменьшить. Чтобы вернуть исходный масштаб выберете пункт меню Вид - в обычном размере.

Перемещение рабочей области

Для перемещение рабочей области(вида) удерживайте кнопку колесика мыши и двигайте мышь.

Сохранение изображния рабочей области

Для сохранения изображения рабочей области выберете пункт меню Инструменты - сохранить изображение. В диалоговом окне можно изменить имя скачиваемого файла(по-умолчанию новый граф) и сохранить, или открыть изображение в новой вкладке. Изображение сохраняется в фомате PNG.

Режимы

Режимы работы программы:

  • Режим вершин
  • Режим ребер
  • Удалить
  • Выделение

Режим вершин - используется по-умолчанию. При клике ЛКМ на рабочей области добавляется новая вершина. Основной режим работы программы, все действия можно выполнять из него. Остальные два режима нужны для удобства.

Режим ребер. При клике ЛКМ на вершине начинается добавление нового ребра, при последующем клике на конечной вершине создается ребро. Данный режим будет полезен, если нужно добавить много ребер.

Удалить. При клике ЛКМ по вершине или ребру, элемент удаляется. Данный режим будет полезен, если нужно удалить много элементов(вершин или ребер).

Выделение. При клике ЛКМ по вершине или ребру, элемент выделяется. Удерживайте Ctrl для добавления элемента к группе выделенных. Возможно выделение множества элементов с помощью мыши. Для этого кликните по пустой области и, удерживая ЛКМ, перемещайте мышь. Данный режим будет полезен, если нужно работать с множеством элементов(вершин или ребер). Для работы с группой выделенных элементов используйте пункт меню Правка.

Настройки

Для изменения настроек выберете пункт меню Правка - настройки. Настройки сохраняются в браузере. Можно настроить именование вершин(арабскими цифрами или латинским алфавитом) и палитру(настраиваются все цвета, использующиеся при отрисовке графа).
